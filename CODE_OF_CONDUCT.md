# Code of Conduct

## Our Pledge

我们致力于为所有参与者提供一个友好、安全、尊重的协作环境。

## Expected Behavior

- 尊重不同经验与观点
- 提供建设性反馈
- 关注问题本身，避免人身攻击
- 接受合理的项目维护决策

## Unacceptable Behavior

- 侮辱、歧视、骚扰性言论
- 威胁、恶意攻击、挑衅
- 泄露他人隐私信息
- 任何影响社区安全与协作的行为

## Enforcement

维护者有权对违反行为准则的内容进行删除、警告、临时或永久禁言/封禁。

如遇问题，请通过 `SECURITY.md` 中的联系方式私下联系维护者处理。
