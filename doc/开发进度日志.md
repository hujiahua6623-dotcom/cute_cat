## 2026-03-21（后端：周期 0 + 周期 1 切片实现）

### 今日完成

1. 新增 `backend/` 工程：`FastAPI` + `SQLAlchemy` 2 异步（`asyncmy` / 测试用 `aiosqlite`）、`Alembic` 初版迁移（`users` / `gardens` / `pets` / `refresh_tokens`）。
2. 认证：`POST /api/v1/auth/register|login|refresh|logout`；access JWT；refresh 存 MySQL（无 Redis 亦可旋转）。
3. 业务：`GET /api/v1/me`，`POST /api/v1/pets/claim`，`GET /api/v1/pets/{petId}`（含被动衰减与简化 `stability`），`GET /api/v1/offline-summary`，`GET /api/v1/gardens/ws-ticket`。
4. 实时：`/api/v1/ws/garden?ticket=`，`joinGarden` / `updatePointer` / `petAction`（`Feed`/`Cuddle`/`Pat`），广播 `gardenSnapshot`、`pointerUpdate`、`actionBroadcast`、`petStateDelta`。
5. 自测：`pytest`（`tests/test_health.py`、`tests/test_auth_pets.py`）全部通过。
6. 文档：更新 [`doc/后端开发设计文档.md`](后端开发设计文档.md) §11、`doc/API-后端与前端对接.md`（健康检查双路径、可选 `itemId`、`petStateDelta` 约定、联调环境变量）、[`README.md`](../README.md)、[`doc/项目目录说明.md`](项目目录说明.md)；根目录 [`.env.example`](../.env.example) 增加 `PUBLIC_BASE_URL`、`WS_TICKET_TTL_SECONDS`。

### 变更文件 / 模块（摘要）

- `backend/src/cute_cat/`：应用入口、`api/routes/*`、`auth/*`、`game/*`、`persistence/*`、`realtime/*`、`services/pet_state.py`
- `backend/alembic/`：初始迁移 `001_initial.py`
- `backend/tests/`、`backend/README.md`、`backend/pyproject.toml`

### 已知风险 / 注意

- `pip install -e .` 若遇本地 pip/setuptools 异常，可用 `PYTHONPATH=src` 运行 `uvicorn` / `pytest`（见 `backend/README.md`）。
